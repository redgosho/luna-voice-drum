# ルナボイスドラム
カグヤルナのボイスでドラムを叩きたかったから作ってみた。
画面に表示される丸いボタンをタップすると、ルナちゃんのボイスが流れる。

## 課題
* Androidでホーム画面に追加の通知が出てこない。
* iOSでホーム画面に追加したときに音が流れない。
* 音の遅延がやばすぎて、ドラムとして機能しない。

url:https://luna-voice-drum.web.app/
